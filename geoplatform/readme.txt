Ceci est un jeu de platforme nommé geoplatform.

---------------------------------------------------------------------------------------------------

Description :
#
# Vous incarnez dans ce jeu un petit carré qui doit arriver à son objectif, mais des obstacles comme des pics se dressent devant lui !
#
# La seule chose que vous pouvez faire est de sauter par dessus ces obstacles...
#
# Bonne chance !

---------------------------------------------------------------------------------------------------

Commandes :

#
# Vous pouvez changer les commandes suivantes dans le menu des paramètres :
#
 # 'sauter' : <espace> par défaut
 # 'pause' : <p> par défaut
 # 'paramètres' : <r> par défaut
#
# Les commandes suivantes sont en revenche invariables :
#
 # 'arcade' : <a>
#

---------------------------------------------------------------------------------------------------
